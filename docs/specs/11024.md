# Spec: Markdown Hyperlinks in Text Elements (#11024)

## Goal
Detect markdown-style links [label](url) in text elements
and render them as clickable hyperlinks on the canvas.

## Context
- Issue: #11024
- Current behavior: [label](url) renders as plain text
- Expected: renders as a clickable hyperlink

## Component / Module
- `packages/excalidraw/element/textElement.ts`
- `packages/excalidraw/renderer/renderElement.ts`

## Non-goals
- Do NOT support full Markdown rendering (bold, italic, etc.)
- Do NOT modify existing link/URL element behavior
- Do NOT change text editing UX

## Acceptance Criteria
1. [label](url) in text is rendered as clickable "label"
2. Click opens url in a new browser tab
3. Links are visually distinct (underline + color)
4. Plain text without links renders unchanged
5. Existing tests pass (yarn test)
6. Build passes (yarn build)
